# tdm-utils

Regroup functions for TDM apps

## Installation

Using npm :

```shell
$ npm i -g tdm-utils
$ npm i --save tdm-utils
```

Using Node :

```js
/* require of tdm-utils module  */
const utils = require('tdm-utils');
```

## Launch tests

```shell
$ npm run test
```

## Build documentation

```shell
$ npm run docs
```
